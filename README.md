# Attendance & Results Management System

A full-stack web application with role-based dashboards for students, faculty, and admin to manage attendance and academic results.

## Features
- Student signup with OTP verification and forgot password flow
- Faculty dashboard: post attendance, mid-exam & semester results
- Student dashboard: view personal attendance and results
- Admin dashboard: approve/decline users, view reports, send absence emails
- Monthly and annual attendance reports with print functionality
- Role-based access control for Students, Faculty, and Admin

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React.js, HTML, CSS, Bootstrap, JavaScript |
| Backend | Node.js, Express.js |
| Database | PostgreSQL (pgAdmin4) |
| Tools | VS Code, REST APIs |

## How It Works
1. Students sign up and get verified by admin
2. Faculty logs in and marks attendance per session
3. Faculty uploads mid-exam and semester results
4. Students view their attendance percentage and results anytime
5. Admin can generate monthly/annual reports and email absentees
